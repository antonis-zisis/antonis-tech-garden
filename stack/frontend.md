# Frontend Tools

- [React](https://reactjs.org/): A declarative JavaScript library for building UIs.
- [React Native](https://reactnative.dev/): Create native apps for Android, iOS, and more using React.
- [NextJS](https://nextjs.org/): The React Framework for the Web.
- [React Router](https://reactrouter.com/): Router for React applications.
- [Vue](https://vuejs.org/): Progressive framework for building user interfaces.
- [Tailwind CSS](https://tailwindcss.com/): Utility-first CSS framework.
- [Zustand](https://zustand.docs.pmnd.rs/getting-started/introduction): A small, fast, and scalable bearbones state management solution.
- [Vite](https://vite.dev/): Next Generation Frontend Tooling.
